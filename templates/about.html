About Capture Cork

Your guide to the best photography spots in Cork, Ireland.

What is Capture Cork?

Capture Cork is a web application designed to help photographers discover, share, and plan visits to photography locations around Cork, Ireland. Whether you're a beginner or a professional, Capture Cork gives you the tools to find the perfect spot for your next shoot.

Features

  • Locations: Browse and add photography locations around Cork with map integration
  • Sun Times: See sunrise and sunset times for each location
  • Tide Data: Check tide times for coastal locations
  • Weather: Real-time weather forecasts for every spot
  • Photo Analysis: Upload photos to analyse dominant colours
  • Trips: Plan multi-location photography trips with route maps
  • Events: Create and join photography events
  • Visits: Track which locations you've been to
  • Reviews: Rate and review locations to help other photographers
  • AI Assistant: Get photography advice and location suggestions

About This Project

Capture Cork was built as a Final Year Project. It uses Flask, PostgreSQL, Google Maps, and various APIs to provide real-time data for photographers exploring Cork.

Note: This service currently only covers photography locations in Cork, Ireland.

Built With
  • Python / Flask
  • PostgreSQL
  • Google Maps API
  • Cloudinary
  • Bootstrap 4
  • Groq AI
